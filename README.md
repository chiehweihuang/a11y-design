# 讓更多人能用的設計（無障礙網頁設計）

**https://chiehweihuang.github.io/a11y-design/**

## 為什麼做這個

全球 95.9% 的網站有無障礙錯誤（WebAIM 2025）。大部分人不是不願意做，而是不知道要做、不知道能幫助到誰。

這份指南不是教科書，是一個人用了 AI 做無障礙之後的實作分享。裡面有 10 個沉浸式模擬讓你體驗不同使用者的處境、12 題情境式測驗、從 30 秒就能開始的檢核清單，以及工具總覽。

## 內容

- **為什麼** — 為什麼要做無障礙、AI 能做什麼不能做什麼
- **角色** — 你的日常工作如何影響無障礙
- **同理心體驗** — 10 個沉浸式模擬（低視力、色盲、螢幕閱讀器、肢體障礙、鍵盤操作、ADHD、自閉症、閱讀障礙、聽覺障礙、焦慮症）
- **知識測驗** — 12 題情境判斷題
- **動手做** — 從 30 秒到 30 分鐘的檢核清單
- **專案規劃** — 團隊協作的五大階段

## 技術

單一 HTML 檔案，零依賴。支援深色模式。基於 WCAG 2.2 AA。
